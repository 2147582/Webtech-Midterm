Search and Rescue Application

This web application allows an individual to report an incident related to calamities like landslides, flashfloods, and others. Reported incidents will be marked as top priority for rescue operations.
The goal of this web application is to allow emergency response agencies to monitor and track down incidents sent by every individual who uses this web application.
